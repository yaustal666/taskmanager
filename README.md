1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/yaustal666/taskmanager
   ```
2. Перейдите в папку проекта:
    ```bash
    cd taskmanager
    ```
3. Скачайте зависимости:
    ```bash
    cd server
    npm i
    ```

    ```bash
    cd client
    npm i
    ```
4. Запуск приложения:
    ```bash
    cd server
    npm run dev
    ```
    ```bash
    cd client
    npm run dev
    ```
